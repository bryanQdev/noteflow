# Gestión del proyecto

## Metodología

El desarrollo de NoteFlow se gestiona con un tablero Kanban en Trello.
Cada funcionalidad principal tiene su propia tarjeta que se mueve entre
columnas según su estado de desarrollo.

## Columnas del tablero

- **Backlog**: funcionalidades definidas pero no iniciadas
- **Todo**: funcionalidades priorizadas para el sprint actual
- **In Progress**: en desarrollo activo
- **Review**: terminadas, pendientes de revisión y pruebas
- **Done**: completadas y verificadas
